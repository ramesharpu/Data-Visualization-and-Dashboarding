# 1.3 Choosing the Right Chart Type

---

## Why Chart Selection Matters?

Wrong chart → Wrong interpretation.

Correct chart → Better insight.

---

## Bar Chart

Used for comparison.

### Example

| Product | Sales |
| ------- | ----- |
| A       | 100   |
| B       | 250   |
| C       | 400   |

Best Visualization:

Bar Chart

Question Answered:

Which product sells most?

---

## Line Chart

Used for trends over time.

### Example

Monthly sales for 12 months.

Question Answered:

Is sales increasing or decreasing?

---

## Pie Chart

Used for proportions.

Example:

Market Share

| Company | Share |
| ------- | ----- |
| A       | 50%   |
| B       | 30%   |
| C       | 20%   |

---

## Scatter Plot

Used for relationships.

Example:

Advertising Spend vs Revenue

Question:

Does higher advertising increase revenue?

---

## Histogram

Used for distribution.

Example:

Employee ages.

Question:

How are employees distributed across age groups?

---

## Map Visualization

Used for geographical analysis.

Example:

State-wise sales performance.

Question:

Which state generates highest revenue?

---

## Chart Selection Matrix

| Objective    | Chart Type   |
| ------------ | ------------ |
| Compare      | Bar Chart    |
| Trend        | Line Chart   |
| Distribution | Histogram    |
| Relationship | Scatter Plot |
| Proportion   | Pie Chart    |
| Geography    | Map          |
